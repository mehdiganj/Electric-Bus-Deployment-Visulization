Electric Bus Deployment Visualization

This project is done as one of the class projects for Advance Data Visualization (CS 6965) Fall 2021 at the University of Utah. The clients of this project are domain scientists (Cathy Liu, Jianli Chen, Yirong Zhou from the Department of Civil & Environmental Engineering) who work on smart city design.

The data for this project is provided by the client in which a spatio-temporal analytical framework i developed to assist transit agencies in identifying the optimal deployment for the BEB system. The BEB deployment problem is optimized considering cost and environmental equity for disadvantaged population. The input of the optimization prblem is the budget and the output is:

  1. Locations and number of both in-depot and on-route charging stations
  2. Number of buses that were to be replaced
  3. The exact buses that were to be replaced

The optimization problem is solved for three different budgets as the followings:

Plan # Plan 1 Plan 2 Plan 3
Budget ($ M) 25 60 120

In this project we provided a visualization panel for the result of the optimization problem for each plan.
