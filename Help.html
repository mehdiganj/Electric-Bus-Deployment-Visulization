User Interaction

The user would select one of the plans (Plan1, Plan2, or Plan3) as desired. The BEB route information table, BEB information, and charging station lists are updated based on the plan. Next, the user can control the time using the time bar at the top of the page. As the user controls the bar, the information on the page on the map is updated based on the information provided by the client. The user also can search for specific information in the table, select a BEB to see its battery status and mileage. On the other hand, they can choose a particular charging station to observe the energy consumption of the charging station during the day.

Data interpretation

The interface provides several aspect of interpretation about data in each section

Inoformation on the map:

The map component displays the status on electric buses, routes, and charging stations on an interactive map where the user can easily visualize information.

BEB route information section:

This component illustrates the information of all the BEBs in each plan. The table contains a list of the BEBs, state of charge (SOC), Mileage, Route and Status of each BEB.

BEB information section:

This section provide the SoC and Mileage of each BEB over the time. The user can select a BEB using a drop-down button to obverve this information.

Charging station section:

This section visualizes the energy usage of each charging station ove time as well as the status of the station in the recent time.
